# Landing Page - Consultoria em Posicionamento e Marca Digital

## Como rodar
Basta abrir o arquivo `index.html` no navegador.

## Onde editar
- Textos e seções: `index.html`
- Cores/estilo: `assets/styles.css`
- Link do WhatsApp e mensagem padrão: `assets/script.js` (variáveis `WHATSAPP_NUMBER` e `DEFAULT_MESSAGE`)

## Dica
Troque "Sua Marca" pelo nome real do seu negócio no topo e no rodapé.
